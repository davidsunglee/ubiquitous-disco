# Dedicated Special Input

Character Specials use a dedicated `special` input and are serialized/replayed like Jump, Tele-Dash, and Strike. This adds a fourth action to keyboard, touch, protocol, and replay surfaces, but it makes cooldown Specials reliable and readable compared with chorded or context-only triggers.

## Considered Options

- Dash plus Strike chord: rejected because precise chord timing is unreliable and hard to read.
- Context-only auto-fire: rejected because it reduces player control and increases input parsing ambiguity.
